Rain Typing Simulator 打包教程

这个项目是 Windows C++ 桌面程序，使用 clang++ 和 windres 编译。下面是完整打包步骤。

一、准备环境

需要一台 Windows 电脑，打开 PowerShell，先安装 LLVM-MinGW：

winget install MartinStorsjo.LLVM-MinGW.UCRT

安装完成后，重新打开 PowerShell。

二、进入源码目录

cd projects
ain-typing-simulator\source

三、允许当前窗口运行脚本

Set-ExecutionPolicy -Scope Process Bypass -Force

四、执行打包脚本

.uild_cpp.ps1

五、输出位置

成功后会生成：

projects/rain-typing-simulator/outputs/淋雨.exe

常见问题

找不到 clang++ 或 windres：重新安装 LLVM-MinGW，然后重启 PowerShell。

禁止运行脚本：执行 Set-ExecutionPolicy -Scope Process Bypass -Force。

下载 exe 被提示风险：个人编译的 Windows 程序常见情况，可以选择保留，或自行编译。

手动编译命令

windres resource.rc -O coff -o resource.o
clang++ -std=c++17 -O2 -municode -mwindows -static main.cpp resource.o -o ..\outputs\淋雨.exe -lcomctl32 -lpsapi -lwinmm -lshell32 -lcomdlg32 -ldwmapi -lgdi32 -lgdiplus -lole32 -ladvapi32 -lwinhttp -liphlpapi -limm32
